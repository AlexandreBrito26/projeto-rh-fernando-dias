# 📌 Portfólio RH — Fernando Dias

Portfólio profissional em formato de **site**, desenvolvido com **HTML + CSS + JavaScript**, com foco em apresentar o perfil do Fernando Dias (RH) de forma moderna, organizada e com “cara de empresa”.

---

## ✨ Visão Geral

Este projeto foi criado para servir como **cartão de visita digital** para a área de **Recursos Humanos**, destacando:

- ✅ Apresentação profissional (resumo e objetivo)
- ✅ Skills e competências-chave
- ✅ Projetos de RH (modelos e rotinas simuladas)
- ✅ Experiência e formação
- ✅ Contato rápido e mensagem pronta para recrutadores

---

## 🧩 Funcionalidades

### ✅ Interface e Navegação
- Menu fixo no topo com navegação por seções
- Scroll suave (smooth scroll) com ajuste de altura do header
- Destaque automático do item de menu conforme a seção atual

### 🌗 Tema Claro/Escuro
- Botão para alternar entre **modo claro** e **modo escuro**
- Preferência do tema salva no navegador (localStorage)

### 🎨 Efeitos Visuais (Leves e Profissionais)
- Fundo com gradiente animado (bem leve, sem pesar)
- Particles reais com Canvas (30 a 60 bolinhas) com linhas sutis
- Transições suaves + animação de entrada ao rolar a página (scroll reveal)

### 🔢 Animação de Números
- Contadores animados na área de “Resumo rápido”
- Inicia automaticamente quando o usuário rola até a seção

### 📦 Modal “Em breve”
- Botões de download nos projetos abrem um modal
- Útil enquanto os arquivos (PDF/planilhas) ainda não estão disponíveis

### 📩 Contato Inteligente
- Caixa com mensagem pronta para recrutadores
- Botão para **copiar mensagem** com 1 clique
- Botão para enviar por e-mail com texto já preenchido
